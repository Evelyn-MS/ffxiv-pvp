# FFXIV Frontline Tracker

A simple web tool that shows the current Frontline battlefield rotation in Final Fantasy XIV.

> ⚠️ Daily reset is based on 15:00 UTC (automatically adjusted to New York time)

## 🔗 Live Demo
https://evelyn-ms.github.io/ffxiv-pvp/

## ⚙️ Features
- Current battlefield (real-time)
- Next battlefield after reset
- Daily reset countdown
- Dynamic background per map
- Clean UI

## 🛠️ Tech
- HTML
- CSS
- JavaScript

## 📌 Notes
Based on Patch 7.5 (Dawntrail Frontline rotation).
